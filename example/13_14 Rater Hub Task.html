    Query Classification for Apps & Games Search
    Understanding Apps & Games Search queries is the critical first step for rating Apps & Games Search tasks. Please assume that the user has visited a mobile app store and is typing into the search box inside the Apps & Games section.

    You will be asked to categorize the query into one of the following types, based on your understanding of the user intent.

    Important: It's important to research the intent of the query first. Web Search is a useful tool to research the intent of the query.

    Query Type
    Description
    Examples
    Specific App
    The query is for a specific app.
    • [facebook] — It's for a specific app Facebook.
    • [pokemon go] — It's for a specific game Pokemon Go.
    Category of Apps
    The query is for apps in a specific category or apps with a specific feature.
    • [rpg] — It's for games in the RPG category.
    • [find houses for rent] — It's for apps with a feature to find houses for rent.
    Series of Apps
    The query is for apps in a specific series.
    • [final fantasy] — It's for a games in Final Fantasy series. 
    Developer of Apps
    The query is for apps developed by a specific developer.
    • [apps by facebook] — It's for apps developed by Facebook, Inc.
    • [ketchapp] — It's for games developed by the game developer Ketchapp.
    Other Intent
    The user intent for this query is clear, but does not fit into any of the types above.
    • [update apps] — The user intent is to update apps installed on their device.
    • [google play card] — The user intent is to acquire or redeem a Google Play gift card rather than an app.
    Not Interpretable
    It's impossible to determine the user's intent from the query.
    • [a]
    • [qfzrszegn]

    Note for queries with multiple interpretations: If a query has multiple interpretations, choose a query type based on the dominant or most common interpretation. For example:
    • [facebook] — It's a name of a specific app Facebook. It's also a name of a developer Facebook, Inc. which develops multiple apps like Facebook and Messenger as well. However, since the dominant interpretation of the query [facebook] would be the specific app Facebook, the appropriate query type is Specific App. If the query were [apps by facebook], the appropriate query type would be Developer of Apps.
    • [final fantasy] — It's a name of a game series Final Fantasy series. It's also a name of a specific game Final Fantasy, which is the first game in Final Fantasy series. However, the first game in Final Fantasy series is more often called as "Final Fantasy 1". The major interpretation of the query [final fantasy] is Final Fantasy series and the appropriate query type is Series of Apps.
    Needs Met Rating for Apps & Games Search
    Rating
    Description
    FullyM
    This rating only applies when a query is looking for a Specific App and the result is the exact specific app.
    HM
    The result meets the needs of and is very satisfying for many users.

    HM results clearly fit the query and are appealing to many users. HM results often have some or all of the following desirable characteristics of apps: have good reputation, offered by an authoritative developer or have visually appealing screenshots.

    The standards for the HM rating is high. This is particularly important for Category of Apps queries with many on-topic results.

    IMPORTANT: For a Specific App seeking query, please do not use this rating.
    MM
    The result meets the needs of and is satisfying for a reasonable fraction of users.

    MM results reasonably fit the query but it may not appeal to as many users as HM results. MM results may have less desirable characteristics of apps than HM results.

    IMPORTANT: For a Specific App seeking query, please use this rating only when a result is a very related app or a very reasonable alternative app to the searched app from an official source and a result would appeal to a reasonable fraction of users. An official source is often the same developer as the searched app. For example:
    • For a query [facebook], Messenger by Facebook is a very related app to the Facebook app from an official source.
    • For a query [facebook], Facebook Lite is a very reasonable alternative app to the Facebook app from an official source.
    Note that MM is not always appropriate when a result is offered by the same developer as the searched app. A result should appeal to a reasonable fractions of users. For example:
    • For a query [facebook],  Facebook Ads Manager appeals only to a small fraction of users because it's an app for businesses which run ads on Facebook. SM would be a more appropriate rating even if it's from an official source.
    SM
    The result meets the needs of and is satisfying for a small fraction of users.

    This rating is also appropriate when it serves a minor interpretation of the query.

    IMPORTANT: Please use this rating when an app is not exactly what a query looks for but the app is a related app or a reasonable alternative app to the searched app(s) and a result would appeal to a small fraction of users. For example:
    • For a query [pokemon go], Poke Genie - Safe IV Calculator is a related app to Pokémon GO because it's a tool app to play Pokémon GO better.
    • For a query [netflix], Hulu is a reasonable alternative app to Netflix because both apps have the similar purpose of watching streaming videos.

    FailsM
    The result meets the needs of and is satisfying for no or almost no users.

    Rating foreign results
    There are apps which don't have descriptions in a language of your locale. For example, this app 微信读书 doesn't have a description in English. In such a case, you'll see a Google Translate button, use it to understand the description.

    Then assign a Needs-Met rating based on how useful the app would be for a user in your locale. A result which don't have descriptions in a language of your locale is not necessarily FailsM. For example:
    • Jelly Shift, which looks to be a very simple game, would be enjoyable even if a description is not written in their language.
    • 微信读书, which is an app to read Chinese books, would be useless if a user doesn't understand Chinese.
