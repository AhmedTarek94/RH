Tab Management System Test

Test Overview

This test verifies the tab management functions implemented in background.js.

The extension should automatically redirect from task/index pages to the main monitoring page.

Function: isTaskIndexTab(url)

Test Cases:

  • isTaskIndexTab("https://www.raterhub.com/evaluation/rater/task/index"): Should return true
  • isTaskIndexTab("https://www.raterhub.com/evaluation/rater"): Should return false
  • isTaskIndexTab("https://www.raterhub.com/evaluation/rater/task/show?taskIds=123"): Should return false
  • isTaskIndexTab(null): Should return false

Function: isMainPageTab(url)

Test Cases:

  • isMainPageTab("https://www.raterhub.com/evaluation/rater"): Should return true
  • isMainPageTab("https://www.raterhub.com/evaluation/rater/task/index"): Should return false
  • isMainPageTab("https://www.raterhub.com/evaluation/rater/task/show?taskIds=123"): Should return false

Integration Points

The tab management system integrates with:

  • chrome.tabs.onUpdated - When pages finish loading
  • chrome.tabs.onActivated - When tabs are switched to
  • scanAllTabsForMainPage() - Periodic scanning every 2 seconds
